## 3 C++ network program (ver. OS stream socket)

Code:
```cpp
 void get_URL( const string& host, const string& path )
 {
+  TCPSocket tcp;
+  tcp.connect( Address( host, "http" ) );
+  tcp.write(
+    vector<string_view>( { "GET ", path, " HTTP/1.1\r\nHost: ", host, "\r\nConnection: close\r\n\r\n" } ) );
+  while ( !tcp.eof() ) {
+    string buffer;
+    tcp.read( buffer );
+    cout << buffer;
+  }
 }
```
